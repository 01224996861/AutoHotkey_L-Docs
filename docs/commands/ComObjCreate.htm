ComObjCreate() [AHK_L 53+]

Creates a COM object.

ComObject := ComObjCreate(CLSID)

Parameters

CLSID CLSID or human-readable Prog ID of the COM object to create.

General Remarks

Strictly speaking, ComObject is not an actual COM object, but a wrapper around a COM object which allows the script to use it more naturally.

Related

ComObjGet, ComObjActive, ComObjConnect, ComObjError, CreateObject (MSDN)

Examples

ie := ComObjCreate("InternetExplorer.Application")
ie.Visible := true
ie.Navigate("http://www.autohotkey.net/~Lexikos/AutoHotkey_L/")
