ComObjError() [AHK_L 53+]

Enables or disables notification of COM errors.

ComObjError(Enable)

Parameters

Enable A boolean value (true or false).

General Remarks

Currently there is no mechanism available for scripts to detect or handle specific COM errors. More general errors, such as failure to create an object, retrieve a property or call a method may be detected by checking for an empty or false return value. However, note that empty return values don't always indicate an error.
