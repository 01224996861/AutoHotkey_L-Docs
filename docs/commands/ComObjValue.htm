ComObjValue() [v1.0.91+]

Retrieves the value or pointer stored in a COM wrapper object.

Value := ComObjValue(ComObject)

Parameters

Value A 64-bit signed integer.
ComObject A wrapper object containing a COM object or typed value.

General Remarks

This function would typically be used after determining the type of value contained by the wrapper object.

Related

ComObjType, ComObjCreate, ComObjGet, ComObjActive
