IsLabel

Returns a non-zero number if the specified label exists in the script.

IsLabel := IsLabel(LabelName)

Parameters

LabelName

Type: String

The name of a label.

Return Value

Type: Integer (boolean)

This function returns 1 (true) if the specified label exists, or 0 (false) if not.

Remarks

This function is useful to avoid runtime errors when specifying a dynamic label for Gosub or Goto.

Related

Labels

Examples

#1

if IsLabel("Label")
    MsgBox "Subroutine exists"
else
    MsgBox "Subroutine doesn't exist"
