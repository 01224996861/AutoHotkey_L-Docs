IsObject() [v1.0.90+]

Determines if a value is an object.

OutputVar := IsObject(ObjectValue)

Parameters

OutputVar

The name of the variable in which to store 1 if Object is an object; otherwise 0.

ObjectValue

A object stored in a variable, returned from a function, stored in another object or written directly.

Related

Objects

Examples

object := {key: "value"}

if IsObject(object)
    MsgBox, This is an object.
else
    MsgBox, This is not an object.
