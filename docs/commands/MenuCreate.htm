MenuCreate

Creates a Menu object, which can be used to create and display a menu.

Menu := MenuCreate(EventObj)

Parameters

EventObj

An object with methods to be called when a menu item is selected. If specified, Add's Callback-or-Submenu parameter accepts a method name instead of a label or function name. However, EventObj is not used by items which are given a submenu or function object.

This can be used to share contextual information between methods. For example, if the menu is a context menu, EventObj can contain information about whichever item was right-clicked. The same effect can be achieved (less efficiently) by omitting EventObj, binding each method to the object with ObjBindMethod and passing the result to Add.

Remarks

To retrieve the tray menu, use menu := A_TrayMenu.
