MonitorGetCount

Returns the total number of monitors.

Count := MonitorGetCount()

Parameters

This function has no parameters.

Return Value

This function returns an integer representing the total number of monitors.

Remarks

Unlike the SM_CMONITORS sub-command of SysGet, the return value includes all monitors, even those not being used as part of the desktop.
