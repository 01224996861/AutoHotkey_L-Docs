String

Converts a value to a string.

StrValue := String(Value)

Return Value

The result of converting Value to a string, or Value itself if it is a string.

If Value is a number, the default decimal formatting is used.

If Value is an object, the return value is the result of calling Value.ToString(). If the object has no such method, an exception is thrown. This method is not implemented by default, so must be defined by the script.
